# 🎬 Netflix UI Clone

Projeto inspirado na interface da Netflix, com foco na tela de seleção de perfis e navegação para catálogo de filmes.

---

## 📌 Sobre o projeto

Este projeto foi iniciado a partir de uma Imersão de Desenvolvimento Front-End proporcionada pela plataforma Alura.

A tela de seleção de perfis foi desenvolvida por mim, incluindo estrutura HTML, estilização em CSS, organização e responsividade layout.

A lógica em JavaScript (tema claro/escuro) e a estrutura inicial da tela de catálogo foram fornecidas durante a imersão e posteriormente integradas ao projeto.

---

## 🚀 Funcionalidades

- Seleção de perfis de usuário
- Navegação para tela de catálogo
- Alternância entre tema claro e escuro
- Layout responsivo para diferentes tamanhos de tela

---

## 🛠️ Tecnologias utilizadas

- Visual Code Studios
- HTML5
- CSS3 (Flexbox e responsividade)
- JavaScript

---

## 🎯 Aprendizados

Durante o desenvolvimento, pratiquei e aprimorei:

- Estruturação semântica com HTML
- Organização e reutilização de CSS
- Responsividade com media queries
- Leitura e adaptação de código já existente

---

## ⚠️ Observações

- A tela de seleção de perfis foi desenvolvida por mim como prática de interface e layout.
- Parte do JavaScript (tema claro/escuro) e a base da tela de catálogo foram fornecidas durante a imersão, apenas sofrendo pequenas alterações próprias sobre os conteúdos apresentados na tela.
- O projeto foi adaptado e reorganizado para melhor entendimento e aprendizado.
